# **HR ANALYTICS**

The "HR Analytics" Dashboard is an Interactive data Visulization project designed to help HR teams make data-driven decisions.It provides insights into employee demographics, attrition, performance, and workforce trends to improve retention and organizational planning.
This dashboard transforms raw HR data into meaningful insights using clear visualizations and KPIs.
## FEATURES
- Employee attrition analysis
- Department-wise employee distribution
- Gender and age demographics
- Experience and salary insights
- Interactive filters and KPIs
## Importance of Project
- Human Resource teams handle large amounts of data.
## Without Analytics:
- Problems like high attrition go unnoticed.
- Decisions are based on assumptions, not data
## This project helps:
- Identify problem areas in departments.
- Improve employee retention.
- Support data-driven HR decisions
## **Data Processing Steps:**
This is how the project is built step by step:
### Data Collection:
- HR dataset loaded from CSV / Excel
### Data Cleaning
- Removed missing or incorrect values
- Fixed data types (numbers, categories)
- Standardized column names
### Data Analysis
- Calculated attrition rate
- Grouped employees by department, age, gender
- Identified trends and patterns
### Dashboard Creation
- KPIs created for key metrics
- Charts and visuals added
- Filters enabled for interaction
## Tools & Technologies Used
- Power BI / Tableau / Excel – Dashboard creation
- Python / SQL – Data cleaning & analysis 
- CSV / Excel – Dataset
- GitHub – Project hosting
## Insights from the Project
Some key insights that can be observed:
- Certain departments have higher attrition
- Employees with low experience leave more frequently
- Salary and job role impact retention
- Workforce demographics reveal diversity patterns
These insights help HR teams take preventive actions.
## Conclusion
The HR Analytics Dashboard provides a clear and interactive way to analyze employee data. 
It helps identify attrition patterns and workforce trends, enabling better HR decision-making through data-driven insights.
