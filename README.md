## **HR ANALYTICS**

The "HR Analytics" Dashboard is an Interactive data Visulization project designed to help HR teams make data-driven decisions.It provides insights into employee demographics, attrition, performance, and workforce trends to improve retention and organizational planning.
This dashboard transforms raw HR data into meaningful insights using clear visualizations and KPIs.
## FEATURES
- Employee attrition analysis
- Department-wise employee distribution
- Gender and age demographics
- Experience and salary insights
- Interactive filters and KPIs
